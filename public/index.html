How to export your calendar

  1. Open Google Calendar in your browser.
  2. Click the gear icon (top right) → Settings.
  3. In the left sidebar, click Import & export.
  4. Click Export. A ZIP file will download.
  5. Extract the ZIP — you'll find one or more .ics files.
  6. Upload those files here.
  1. Go to outlook.live.com/calendar or your Outlook Web app.
  2. Click the gear icon → View all Outlook settings.
  3. Navigate to Calendar → Shared calendars.
  4. Under Publish a calendar, select your calendar and choose Can view all details.
  5. Click Publish and copy the ICS link.
  6. Open the ICS link in your browser to download the file.
  7. Upload the downloaded .ics file here.
  1. Open Outlook and go to Calendar view.
  2. Click File → Save Calendar.
  3. Choose Date Range: select at least the last 90 days.
  4. Set Detail to Full details.
  5. Save as .ics file.
  6. Upload the saved file here.
  1. Open Calendar app on your Mac.
  2. In the sidebar, right-click the calendar you want to export.
  3. Select Export…
  4. Choose a save location and click Export.
  5. Upload the .ics file here.
